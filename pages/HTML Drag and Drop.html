    • Arraste e solte!

    Uma página HTML com a funcionalidade de arrastar e soltar (drag and drop) geralmente consiste em elementos que podem ser arrastados e uma área na qual esses elementos podem ser soltos. Aqui está uma explicação dos principais componentes de uma página HTML com drag and drop:

    Elemento Arrastável: Este é um elemento HTML que pode ser arrastado pelo usuário. Para tornar um elemento arrastável, geralmente usamos o atributo draggable="true". Por exemplo:


    COMANDO: <div> id="dragItem" draggable="true">Arraste-me</div>
    Arraste-me


    Área de Soltar: Este é um elemento HTML que atua como uma zona onde os elementos arrastáveis podem ser soltos. Para permitir que um elemento seja solto na área de soltar, precisamos definir eventos para detectar quando um elemento é arrastado sobre a área de soltar e quando ele é solto na área de soltar. Por exemplo:


    COMANDO: <div> id="dropZone" ondrop="drop(event)" ondragover="allowDrop(event)">Solte aqui</div>
    Solte aqui

    Precisamos de JavaScript para controlar o comportamento do arrastar e soltar. Isso inclui definir o que acontece quando um elemento é arrastado, permitir que um elemento seja solto na área de soltar e definir o que acontece quando um elemento é solto na área de soltar. Como exemplo de comando abaixo:


    <script> function drag(event) { event.dataTransfer.setData('text/plain', event.target.id); } function allowDrop(event) { event.preventDefault(); } function drop(event) { event.preventDefault(); var data = event.dataTransfer.getData('text/plain'); var draggedElement = document.getElementById(data); event.target.appendChild(draggedElement); } function highlightDropZone(event) { event.target.classList.add('highlight'); } function unhighlightDropZone(event) { event.target.classList.remove('highlight'); } </script>
