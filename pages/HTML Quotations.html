    • Tags HTML de Citação com Exemplos

    As tags de citação são usadas para marcar citações de texto em uma página HTML. Aqui estão algumas das tags de citação mais comuns:

    <q>

    A tag <q> é usada para marcar citações curtas dentro de um parágrafo. O navegador normalmente adiciona aspas ao texto citado.

    O autor disse: Isso é uma citação curta.

    <blockquote>

    A tag <blockquote> é usada para marcar citações longas que podem se estender por vários parágrafos. Ela define uma seção de texto que deve ser renderizada como uma citação separada do texto circundante.

    Esta é uma citação longa que ocupa vários parágrafos. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus ac nisi velit. Sed sit amet gravida nulla. Integer non tincidunt arcu. Integer sodales orci a justo rhoncus aliquet. Duis at urna in ligula dictum varius.

    Curabitur auctor vestibulum enim, id lacinia nisi luctus at. Sed lobortis est at leo commodo tincidunt. Nulla lacinia, mi non maximus luctus, ipsum elit tristique felis, eu facilisis lorem risus nec dolor.

    <cite>

    A tag <cite> é usada para marcar o título de uma obra citada, como um livro, filme ou artigo.

    O livro O Senhor dos Anéis é uma obra épica de fantasia.
