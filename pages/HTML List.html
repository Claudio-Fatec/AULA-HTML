    • Tags HTML de Listas com Exemplos

    As listas são usadas para organizar informações em formato de lista. Aqui estão alguns exemplos de tags de lista em HTML:

    <ul>

    A tag <ul> cria uma lista não ordenada, onde cada item é marcado por um marcador, geralmente um círculo ou ponto.

    • Item 1
    • Item 2
    • Item 3

    <ol>

    A tag <ol> cria uma lista ordenada, onde cada item é numerado.

    1. Item 1
    2. Item 2
    3. Item 3

    <dl>

    A tag <dl> cria uma lista de definição, onde cada item consiste em um termo (dt) seguido por sua definição (dd).

    Nested Lists (Listas Aninhadas)

    Você pode aninhar listas dentro de outras listas para criar estruturas mais complexas:

    • Item 1
    • Item 2
      • Subitem 2.1
      • Subitem 2.2
    • Item 3
