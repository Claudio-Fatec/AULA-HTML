    • Tags HTML de Atributos com Exemplos

    <a> - href

    O atributo href é usado na tag <a> para especificar o URL para o qual o link aponta.

    Exemplo de Link

    <img> - src, alt

    Os atributos src e alt são usados na tag <img> para especificar o caminho do arquivo da imagem e fornecer um texto alternativo, respectivamente.

    <input> - type, value

    Os atributos type e value são usados na tag <input> para definir o tipo de entrada e o valor padrão, respectivamente.

    <meta> - charset, name, content

    Os atributos charset, name e content são usados na tag <meta> para especificar o conjunto de caracteres, o nome da meta informação e o conteúdo da meta informação, respectivamente.

    <style> - type

    O atributo type na tag <style> é usado para definir o tipo de conteúdo do estilo. Normalmente é definido como "text/css" para indicar que o estilo contém CSS.

    <script> - src, type

    Os atributos src e type na tag <script> são usados para especificar o URL do arquivo de script e o tipo de conteúdo do script, respectivamente.
