Entidades HTML com Exemplos

As entidades HTML são códigos especiais usados para representar caracteres especiais ou reservados em HTML. Aqui estão alguns exemplos:

Entidades de Caracteres Especiais

Estas entidades são usadas para representar caracteres que têm significados especiais no HTML:

  • < representa o caractere menor que (<)
  • > representa o caractere maior que (>)
  • & representa o caractere e comercial (&)
  • " representa o caractere de aspas duplas (")
  • ' representa o caractere de aspas simples (')

Entidades de Caracteres Acentuados

Estas entidades são usadas para representar caracteres acentuados ou especiais:

  • á representa a letra á (á)
  • é representa a letra é (é)
  • í representa a letra í (í)
  • ó representa a letra ó (ó)
  • ú representa a letra ú (ú)
  • Á representa a letra Á (Á)
  • É representa a letra É (É)
  • Í representa a letra Í (Í)
  • Ó representa a letra Ó (Ó)
  • Ú representa a letra Ú (Ú)
