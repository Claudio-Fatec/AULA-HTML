    • Tags HTML de Tabela com Exemplos

    As tabelas HTML são usadas para exibir dados em filas e colunas. Elas são compostas principalmente pelas seguintes tags:

    • <table>: Define a tabela.
    • <tr>: Define uma linha na tabela.
    • <th>: Define um cabeçalho de coluna na tabela.
    • <td>: Define uma célula na tabela.

    Exemplo Básico

    Aqui está um exemplo básico de uma tabela:

    Nome Idade Cidade
    João 30 São Paulo
    Maria 25 Rio de Janeiro

    Tabela com Estilos

    Você pode estilizar a tabela usando CSS:

    Nome Idade Cidade
    Carlos 35 Brasília
    Ana 28 Salvador

    Tabela com Colspan e Rowspan

    Você também pode mesclar células em uma tabela usando os atributos colspan e rowspan:

    Nome Informações
    Carlos Informações Adicionais:
    Idade: 35
    Cidade: Brasília
    Ana
