    • Tag HTML SVG com Exemplos

    A tag <SVG> é usada para desenhar gráficos vetoriais escaláveis na web. Ela oferece uma maneira poderosa de criar gráficos, animações e elementos interativos. Aqui estão alguns exemplos básicos de como usar a tag <SVG>:

    Exemplo 1: Retângulo

    <SVG> width="200" height="100"> <rect&g x="50" y="20" width="100" height="50" fill="blue" /> </SVG>

    Exemplo 2: Círculo

    <SVG> width="200" height="200"> <circle&g cx="100" cy="100" r="50" fill="green" /> </SVG>

    Exemplo 3: Linha

    <SVG> width="200" height="200"> <line&g x1="50" y1="50" x2="150" y2="150" stroke="red" /> </SVG>

    Exemplo 4: Texto

    <SVG> width="200" height="100"> <text> x="50" y="50" fill="black">Olá, SVG!</text> </SVG>
