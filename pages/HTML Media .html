Tags HTML de Mídia com Exemplos

As tags <audio> e <video> em HTML são usadas para incorporar áudio e vídeo em páginas web. Aqui estão alguns exemplos de como usar essas tags:

Tag <audio> : Reproduzindo Áudio

Este exemplo usa a tag <audio>` para reproduzir um arquivo de áudio:

<audio> controls> <source> src="/AULA-HTML/darth_vader.mp3" type="audio/mp3"> Seu navegador não suporta o elemento de áudio. </audio>

Tag `<video>: Reproduzindo Vídeo

Este exemplo usa a tag <video> para reproduzir um arquivo de vídeo:

<video> controls width="400"> <source> src="/AULA-HTML/darth.mp4" type="video/mp4"> </video>
