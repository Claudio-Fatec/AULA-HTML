Tags HTML Básicas

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ele usa tags para estruturar o conteúdo. Aqui estão algumas tags básicas:

<html>

A tag <html> define o início e o fim de um documento HTML.

<head>

A tag <head> contém informações sobre o documento, como título, meta informações e links para estilos.

<title>

A tag <title> define o título do documento, que é exibido na barra de título do navegador.

<body>

A tag <body> contém o conteúdo visível da página, como texto, imagens, links, etc.

<h1>, <h2>, ..., <h6>

As tags de cabeçalho definem títulos de diferentes níveis, do mais importante (<h1>) ao menos importante (<h6>).

<p>

A tag <p> define um parágrafo de texto.

<a>

A tag <a> define um hiperlink para outra página ou recurso.

<img>

A tag <img> exibe uma imagem na página.

<ul>, <ol>, <li>

As tags <ul> e <ol> criam listas não ordenadas e ordenadas, respectivamente. A tag <li> define cada item da lista.

  • Item 1
  • Item 2
  • Item 3

<div>

A tag <div> é usada para agrupar elementos em um bloco e aplicar estilos.

<span>

A tag <span> é usada para aplicar estilos ou scripts a parte do texto dentro de um elemento.
