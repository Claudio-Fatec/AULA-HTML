Tags HTML de Cores com Exemplos

As cores podem ser especificadas em HTML usando diferentes métodos. Aqui estão alguns exemplos:

Cor de Texto

Para definir a cor do texto, use o atributo style com a propriedade color:

Este texto está vermelho.

Cor de Fundo

Para definir a cor de fundo de um elemento, use o atributo style com a propriedade background-color:

Este é um exemplo de fundo com cor azul claro.

Cor de Borda

Para definir a cor da borda de um elemento, use o atributo style com a propriedade border-color:

Este é um exemplo de borda com cor verde.

Cores por Nome

Algumas cores podem ser especificadas por nome, como red, blue, green, etc.:

Este texto está azul.

Cores em Código Hexadecimal

As cores também podem ser especificadas em código hexadecimal, como #RRGGBB, onde RR representa a intensidade de vermelho, GG representa a intensidade de verde e BB representa a intensidade de azul:

Este texto está vermelho (#ff0000).

Cores em Código RGB

As cores podem ser especificadas em código RGB, como rgb(r, g, b), onde r, g e b representam as intensidades de vermelho, verde e azul, respectivamente:

Este texto também está vermelho (rgb(255, 0, 0)).
