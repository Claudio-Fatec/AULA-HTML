Tag HTML video com Exemplos

A tag <video> em HTML é usada para incorporar vídeos em uma página web. Aqui está um exemplo básico:

Exemplo 1: Reproduzindo um Vídeo

<video> controls width="400"> <source> src="/caminho do arquivo" type="video/mp4"> Seu navegador não suporta o elemento de vídeo. </video>

Neste exemplo, usamos a tag <video> para incorporar um vídeo em uma página. O atributo `controls` adiciona controles de reprodução (como botões de play/pause, volume etc.).
