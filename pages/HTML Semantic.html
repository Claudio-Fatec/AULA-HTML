Tags HTML de Semântica com Exemplos

As tags de semântica em HTML são usadas para dar significado ao conteúdo da página, melhorando a acessibilidade e a indexação pelos mecanismos de busca. Aqui estão alguns exemplos de como usar tags de semântica:

<header>

Meu Site

Este é o cabeçalho do meu site.

<nav>

  • Página Inicial
  • Sobre
  • Contato

<section>

Seção 1

Esta é a primeira seção do conteúdo.

<article>

Artigo 1

Este é um artigo sobre um determinado assunto.

<aside>

Barra Lateral

Este é um conteúdo adicional na barra lateral.

<footer>

Rodapé do site.
