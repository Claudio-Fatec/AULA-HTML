    • Tags HTML de Elementos com Exemplos

    HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Aqui estão algumas tags de elementos comuns:

    <div>

    A tag <div> é usada para criar divisões ou seções em um documento HTML. É frequentemente usada como um contêiner genérico para agrupar elementos e aplicar estilos.

    Este é um exemplo de div.

    <span>

    A tag <span> é usada para aplicar estilos ou scripts a parte do texto dentro de um elemento. Ela geralmente é usada para aplicar estilos a partes específicas de um texto.

    Este é um exemplo de span dentro de um parágrafo.

    <header>

    A tag <header> define o cabeçalho de uma seção ou página. Pode conter logotipos, menus de navegação, etc.
