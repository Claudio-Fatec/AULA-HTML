    • Tags HTML de Formatação com Exemplos

    As tags de formatação são usadas para aplicar formatação especial ao texto em uma página HTML. Aqui estão algumas das tags de formatação mais comuns:

    <strong>

    A tag <strong> é usada para destacar texto importante. O navegador normalmente renderiza o texto em negrito.

    Este texto está em negrito.

    <em>

    A tag <em> é usada para enfatizar o texto. O navegador normalmente renderiza o texto em itálico.

    Este texto está em itálico.

    <u>

    A tag <u> é usada para sublinhar o texto.

    Este texto está sublinhado.

    <sup>

    A tag <sup> é usada para exibir texto como sobrescrito.

    x<sup>2</sup>

    <sub>

    A tag <sub> é usada para exibir texto como subscrito.

    H<sub>2</sub>O

    <mark>

    A tag <mark> é usada para destacar partes do texto com uma cor de fundo diferente.

    Este é um exemplo de texto com destaque.
