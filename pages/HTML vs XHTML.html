    • HTML vs XHTML

    HTML (Hypertext Markup Language) e XHTML (Extensible Hypertext Markup Language) são linguagens de marcação utilizadas para criar documentos na web. Aqui estão algumas diferenças entre elas:

    HTML

    HTML é mais permissivo em relação à sintaxe e formatação dos documentos. Aqui está um exemplo de um documento HTML:

                
<!DOCTYPE html>
<html>
<head>
    <title>Exemplo HTML</title>
</head>
<body>
    <h1>Olá, mundo!</h1>
    <p>Este é um exemplo de HTML.</p>
</body>
</html>

    XHTML

    XHTML é uma reformulação do HTML em XML, o que significa que segue uma sintaxe mais estrita. Aqui está um exemplo de um documento XHTML:

                
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
 "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title>Exemplo XHTML</title>
</head>
<body>
    <h1>Olá, mundo!</h1>
    <p>Este é um exemplo de XHTML.</p>
</body>
</html>

    Enquanto o HTML é mais flexível, o XHTML segue regras mais rígidas de formatação, como a necessidade de fechar todas as tags e utilizar caracteres em caixa baixa para as tags e atributos.
